## 🚀 Deploy to Vercel (3 Steps)

### Step 1: Install Dependencies

```bash
# Install root dependencies (fs-extra for build script)
npm install

# Install Plus1-Rewards dependencies
cd plus1-rewards
npm install
cd ..

# Install Plus1-Go dependencies
cd plus1-go
npm install
cd ..
```

### Step 2: Build Combined Project

```bash
# From root directory
npm run build:all
```

This will:
1. Build Plus1-Rewards → `plus1-rewards/dist/`
2. Build Plus1-Go → `plus1-go/dist/`
3. Combine both into → `dist/`

### Step 3: Deploy to Vercel

```bash
# Install Vercel CLI (if not installed)
npm i -g vercel

# Deploy
vercel --prod
```

Follow the prompts:
- **Project name:** `plus1-ecosystem`
- **Directory:** `./` (root)
- **Override settings?** No

## 🌐 Add Custom Domain

After deployment:

1. Go to Vercel Dashboard → Your Project
2. Navigate to **Settings** → **Domains**
3. Add domain: `www.plus1rewards.com`
4. Follow DNS instructions

### DNS Configuration

Add these records to your DNS provider:

```
Type: A
Name: www
Value: 76.76.21.21
TTL: 3600
```

Or use CNAME:

```
Type: CNAME
Name: www
Value: cname.vercel-dns.com
TTL: 3600
```

## 🔧 Troubleshooting

### Build Fails

**Error:** `Module not found`
```bash
# Solution: Clean and reinstall
cd plus1-rewards
rm -rf node_modules package-lock.json
npm install

cd ../plus1-go
rm -rf node_modules package-lock.json
npm install
```

### Plus1-Go Routes Not Working

**Problem:** `/go/dashboard` returns 404

**Solution:** Ensure `vercel.json` has correct rewrites:
```json
{
  "source": "/go/:path(.*)",
  "destination": "/go/index.html"
}
```

### Assets Not Loading on Plus1-Go

**Problem:** CSS/JS files 404 on `/go` pages

**Solution:** Check `plus1-go/vite.config.ts` has:
```typescript
base: '/go/'
```

### Environment Variables Not Working

**Problem:** `import.meta.env.VITE_*` is undefined

**Solution:**
1. Ensure variables start with `VITE_`
2. Add them in Vercel Dashboard
3. Redeploy after adding variables
