# INSURER - Complete Feature & Activity Guide

**Role:** Insurer (Medical Cover Provider)  
**Platform:** Plus1 Rewards  
**Purpose:** View approved cover plans, receive member data, manage exports

---

## 1. REGISTRATION & ONBOARDING

### 1.1 Initial Registration
- Enter insurer/provider company name
- Enter contact person full name
- Enter mobile number (becomes login username)
- Create 6-digit PIN (becomes login password)
- Enter contact phone number
- Enter email address
- Accept terms and conditions
- Submit registration

### 1.2 Application Review
- Application status: Pending
- Wait for admin approval
- Receive notification when approved/rejected

### 1.3 Account Activation (After Approval)
- Account status changes to Active
- Receive approval notification
- Can now log in
- Access insurer dashboard
- Begin viewing cover plan data

## 7. EXPORT MANAGEMENT

### 7.1 Create Export
**Process:**
1. Select export month
2. Select cover plans to include
3. Select member status (Active/All)
4. Review export summary:
   - Total cover plans
   - Total members
   - Total funded amount
5. Generate export
6. Export status: Pending
7. Admin reviews and approves
8. Export status: Completed
9. Download export file

### 7.2 Export Formats

- Excel (XLSX)
- PDF report
- Custom format (if configured)
